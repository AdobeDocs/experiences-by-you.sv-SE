# Hantera intressenter för att implementera Marketo Engage

Att implementera Marketo Engage är ett kritiskt ögonblick när du utvecklar din MarTech-stack. Det kan omfatta olika intressenter som ni behöver för att ta med er från marknadsföring till försäljning till IT. Lär dig hur du får support från din organisation för din nya instans av Marketo Engage genom att ställa rätt frågor och skicka information om uppdateringar och support som behövs regelbundet. Använd självstudiekursen och mallarna (med nedladdningsbara versioner) för att vägleda den interna kommunikationen genom hela implementeringen och användarintroduktionen.

## Hur ni interagerar med intressenter och ledarskap för implementeringen av Marketo Engage
